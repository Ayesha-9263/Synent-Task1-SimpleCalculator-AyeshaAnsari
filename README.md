## 📖 Project Description
The Simple Calculator is a command-line based application built using Python. It allows users to perform basic arithmetic operations in an interactive way. The project focuses on implementing core programming concepts like functions, conditional statements, and exception handling.

## 🎯 Objective
The main objective of this project is to understand how to take user input, process it using logical conditions, and display accurate results in a structured format.

## ⚙️ Features
- Perform Addition, Subtraction, Multiplication, and Division  
- Handles division by zero  
- Takes user input through terminal  
- Displays results instantly  
- Handles invalid inputs using exception handling  

## 🛠️ Technologies Used
- Python 3  
- Command Line Interface (CLI)
